A New Beginning

发表于 2017-11-03 | | 阅读次数:

  emmmm选择困难症的我，终于选了个还阔以的主题Next，其实是想弄这种高端的style–Miccall，但是搭这个博客的初衷只是记录一下自己学了什么，简陋点也无妨。（别编了，你就是弄不来）
  在去年小白搭他的网站时，我就有也搭一个的想法了。Anyway，总算是开始了，thx心哥提供的idea，附上今天下午刚学的PS，@本叔教的真的很好。

为什么要写

  从初三到高三，断断续续地写了4年日志。那些阳光透过树隙洒下的斑驳岁月，那些路灯投在路面拉长的冰凉身影，那些有的没的，到如今成了没的。那些日志尘封在某个角落，从没翻看过。倘若一火成灰烬，记忆便打碎为残缺。希望站在20层的象牙塔，拾起前19年的匆匆岁月，还不算太晚。

想写什么

  • OneNote里面有很多笔记，不过都是细枝末节的知识点，想好好整理一下
  • 现在在学的操作系统和计网，以及自学的图形学、Java，还想借此自学一下Python
  • 深度学习相信是今后的潮流，弄一弄caffee
  • 想起大一很喜欢的英语老师，她叫Leung，那时每周会看一篇PNAS，后来..淡忘了
  • 还有就是PS啦，不过都是跟着教程学，所以就当是赏心悦目

最最最重要的

  你可能也和我一样懒，懒得买去买服务器，懒得去登记，懒得自己写前端（emmm，这个应该是不会写），Hexo+Github Page你值得拥有。
